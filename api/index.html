Robotic Arm API

The main API for the robotic arm can be access here commands can be sent directly using the following format as a URL; http://127.0.0.1:8080/send?command=motor,angle,other In this case motor represents the motor to move, between 0 and 5, angle the rotation in degrees and other is not yet set. Commands can also be passed directly on this page and log files of received and sent data accessed.
