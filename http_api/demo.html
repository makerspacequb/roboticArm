Robotic Arm Demos

A Selection of demos for the robotic arm that run from the Raspberry Pi connected to the Arm.

Xbox Control

This demo allows control of each joint via an Xbox Controller.

NOTE: This demo currently runs on startup.

Xbox Control via Inverse Kinematics

This demo allows control of the end effector positions via an Xbox Controller.

Standup and Sit Down

This demo calibrates the robot then the robot moves to an upright position before imediately returning to a lying down.

Random Positioning

Calibrates the arm, move the arm to it's upright position and then selects a random combination of joints to move to.

WARNING: This demo may not always operate safely. The combination of random joints may result in object collision.

Move and Return

Calibrates the arm, moves all joints by 45 degrees. The arm then returns to previous it original position.

Status

When commands are sent, their status is shown here.
